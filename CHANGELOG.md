# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-01-16

### Added
- **Keyboard Backlight Control** - Brightness levels (Off/Low/High) and static color presets
- **Privacy Spoiler Toggle** - Discord-style gray boxes to hide sensitive device info
- **CLI Improvements** - `--help` and `--version` flags with usage examples
- **Scrollable GUI** - Better support for smaller screens
- **Admin Elevation Manifest** - Auto-request administrator privileges

### Changed
- Version bump to 0.2.0
- Improved README with screenshot and better documentation

### Fixed
- GUI scrolling issues on smaller displays

## [0.1.0] - 2026-01-14

### Added
- Initial release
- Power profile switching (Quiet/Balanced/Performance)
- Battery Conservation Mode toggle
- Rapid Charge toggle
- Device detection for Legion and LOQ models
- Modern dark-mode GUI
- CLI interface with JSON output support
- Dry-run mode for safe testing
